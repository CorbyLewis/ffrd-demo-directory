# hydrology
## Description:
This folder includes the HEC-HMS model used for the *conformance* phase of modeling.
See Job Aid 13 - Model Delivery for Cloud Compute for certain adjustments to model files required prior to delivery.

### archive folder
The *"archive"* folder is used to house draft and outdated model versions as updates are made. Archived models should be date stamped. Examples of an archived model unit directory name:
> model-unit-YYYYMMDD
> trinity_20250314
