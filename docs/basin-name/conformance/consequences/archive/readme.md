# archive

#consequences
This folder contains *archived* structure inventories that represent earlier versions of the inventory in the *conformance* phase.

The nsi year should reflect the year (version) of the nsi, and the year of the price level should reflect the price level year if prices are indexed, the yyyymmdd repreesents the date of the archive of the inventory.

Naming convention: <modeling-unit>-nsiyyyy-pricelevelyyyy-yyyymmmdd.gpkg
