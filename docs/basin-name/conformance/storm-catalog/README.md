# STORM-CATALOG

This folder contains the storm catalog for hte basin.

The **catalog.grid** files which contain pointers to all storms in the catalog. It also includes the **transposition domain** and **watershed boundary**.

The **storms** subdirectory contains all precipitation and temperature data in .dss file format.

The **fishnets** folder contains the points which are valid placements for each storm.

**Seasonality distributions** folder are the storm typed seasonalities and govern when storms of a given type can occur within the calendar year.
