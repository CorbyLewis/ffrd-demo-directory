# existing-models
This folder contains existing H&H models (e.g. hydraulic, hydrologic, reservoir-operations) to be used as reference during FFRD model development.

These models may be sourced from the following locations (additional model sources beyond this list are likely):
* Corps Water Management System (CWMS) modeling effort
* USACE District Models
* High-Level Risk Assessments
* FEMA models
* Other local and Federal agencies

All models included in this folder should include associated **documentation, reporting, and metadata**.

*Note that non-model FEMA data and reports are saved in a separate "FEMA" directory.*
