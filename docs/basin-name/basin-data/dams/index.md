# dams
## Folder includes:

* Information regarding dams in the study area (e.g., risk assessment data, WCMs, O&M manuals, survey data, as-built drawings, dam watch data, major reservoir’s operational guidance, NRCS data)
* Any reporting of efforts to refine the NID data for the purposes of the FFRD study are also included here.
* If **system-response curves** are generated during the *data preparation* phase, this information can be included within the *dams* folder. This information should be promoted to the later *calibration* or *conformance* phase folders once it has been reviewed and meets the criteria for these later phases.


## NID (National Inventory of Dams) Subfolder
Folder includes static version of the National Inventory of Dams dataset collected during Data Preparation Phase. This 
NID data includes geospatial information.
