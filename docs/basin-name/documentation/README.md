# DOCUMENTATION
The documentation directory is the centralized storage of all work plans, reports, and reviews for the entire project. The documentation will advance from draft stages to final products within this location

Documentation will be divided by both model type (e.g. hydraulics, hydrology, reservoir-operation) and also by workflow phases (e.g. calibration, conformance, production). All aspects of the project should be well documented in this directory, including components like meteorology and consequences.

### Documentation Folder Structure:
- Calibration spreadsheets
- Reports
- Review-checklists
- Work-plans
