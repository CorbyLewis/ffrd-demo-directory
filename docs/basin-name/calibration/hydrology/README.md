# Hydrology
## Description:
This folder contains hydrologic model resulting from the *Calibration* phase. This model is promoted to the *Conformance* folder when review is complete and the *Calibration* phase is closed out.

Models should include all relevant files required to simulate the model on desktop. This includes all model files associated with historic events used for calibration. Any referenced external data should be included.

### Historic-Events Folder
The *historic-events* folder contains event-based hydrologic models where individual calibration/validation events were simulated.

### Continuous-Simulation Folder
The *continuous-simulation* folder contains the period-of-record (POR) simulation hydrologic model.

### Archive Folder
The *archive* folder is used to house draft and outdated model versions as updates are made. Archived models should be date stamped. Examples of an archived model unit directory name:
> basin-name_YYYYMMDD
> 
> trinity_20250314
